Control yeelight with Siri and raspberry pi3

发表于 2017-02-24 | 分类于 智能家居

Yeelight LED Bulb (Color)

iPhone 7

1.Setting the rapsberry pi3.

ssh pi@raspberrypi.local (e.g ssh pi@192.168.10.xxx)

Once you’re logged into your Pi, you should begin by updating the default system packages.

2.Turn on the developer mode by using yeelight App.

3.Install Node.js.

Once done, using node -v command to check the version of Node.js.

4.Install other Dependencies.

5.Install Homebridge and dependencies.

if /user/local/lib No such file or directory.Use /usr/lib/ instead.

6. Install yeelight plugin.

Then open Home App on your iPhone and add the yeelight with code above.
